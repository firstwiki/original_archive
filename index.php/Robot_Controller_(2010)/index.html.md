# Robot Controller (2010)

### From FIRSTwiki

Jump to: navigation, search

The 2010 Robot Controller is the same CompactRIO (cRIO) control system as it
was before from 2009. The cRIO from National Instruments can be programmed in
LabVIEW, C, C++, or Java.
