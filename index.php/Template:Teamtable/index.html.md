## Usage

    
    
    {{teamtable|
    |name=Name
    |color=HTML color
    |image=Image
    |year=Rookie year
    |school=School
    |location=Location
    |website=Website link
    |size=Student Size
    }}
    

This team sidebar is for use on the team's main page only, it must not appear
on any secondary pages about a team.

[[edit](/index.php?title=Template:Teamtable&action=edit&section=2 "Edit
section: Blank table" )]

### Blank table

    {{teamtable|

    |year=
    |school=
    |location=
