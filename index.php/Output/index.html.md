Controller](Robot_Controller "Robot Controller" ) and the [Operator
Interface](Operator_Interface "Operator Interface" ). In general,
[software](Programming "Programming" ) on the
[RC](Robot_Controller "Robot Controller" ) is able to send
different signals through the output ports.

##  Types of Output


### Analog Output

Analog output is a sent as a [PWM signal](PWM_signal "PWM signal" )
between 0 to 254 (the [RC](Robot_Controller "Robot Controller" )
**does not** use the 255 signal).


### Digital Output

Digital output is simply a 0 or 1.
