The 2003 [Robot Controller](/index.php/Robot_Controller "Robot Controller" )
is basically the same as the [2000 RC](/index.php/Robot_Controller_%282000%29
"Robot Controller \(2000\)" ) except that it added support for [Autonomous
mode](/index.php/Autonomous_mode "Autonomous mode" ).

It uses the [BS2sx](/index.php/BS2sx "BS2sx" ) microcontroller, as does the
[Robovation (2003)](/index.php/Robovation_%282003%29 "Robovation \(2003\)" ).
[InnovationFirst.com](http://innovationfirst.com "http://innovationfirst.com"
) has [Documentation for 2001 through 2003 Control
Systems](http://innovationfirst.com/FIRSTRobotics/documentation-legacy.htm
"http://innovationfirst.com/FIRSTRobotics/documentation-legacy.htm" ).
