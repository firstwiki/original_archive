# RadioShack Innovation in Control Award

## From FIRSTwiki

Jump to: navigation, search

This award celebrates an innovative control system or application of control components to provide unique machine functions.

In the past, teams have won this for wearable controls, miniature controlable versions of arms, and detecting regions of the field. This award used to be called "Leadership in Control Award".
